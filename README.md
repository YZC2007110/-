# Python自学笔记
## 项目介绍
本仓库用来存放每日Python学习笔记，学习方向：Python爬虫 + 逆向/n
跟随B站网课系统自学，持续更新学习记录与项目源码

## 目录规划
- `01-basic-syntax`：Python基础语法全量练习（变量、数据类型、分支循环、字符串、列表字典等）
- `02-advanced-core`：Python进阶核心（函数、面向对象、文件操作、异常处理、装饰器）
- `03-spider-basic`：爬虫基础入门（网络请求、数据解析、正则、数据持久化）
- `04-reverse-spider`：爬虫进阶与JS逆向（反爬绕过、异步爬虫、混淆分析、AST还原）
- `05-practice-projects`：实战项目合集（完整爬虫、逆向复现、综合练手项目源码）
